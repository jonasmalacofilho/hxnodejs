Duplex

Duplex streams are streams that implement both the Readable and Writable interfaces.

DuplexNewOptions

Options for Duplex private constructor. For stream implementors only, see node.js API documentation

IDuplex

IReadable

IReadable interface is used as "any Readable".

IWritable

Writable interface used for type parameter constraints. See Writable for actual class documentation.

PassThrough

This is a trivial implementation of a Transform stream that simply passes the input bytes across to the output. Its purpose is mainly for examples and testing, but there are occasionally use cases where it can come in handy as a building block for novel sorts of streams.

Readable

The Readable stream interface is the abstraction for a source of data that you are reading from. In other words, data comes out of a Readable stream.

ReadableEvent

Enumeration of events emitted by the Readable class.

ReadableNewOptions

Options for Readable private constructor. For stream implementors only, see node.js API documentation

Transform

Transform streams are Duplex streams where the output is in some way computed from the input. They implement both the Readable and Writable interfaces.

Writable

The Writable stream interface is an abstraction for a destination that you are writing data to.

WritableEvent

Enumeration for Writable class events.

WritableNewOptions

Options for Writable private constructor. For stream implementors only, see node.js API documentation
